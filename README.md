# Projeto Visão Computacional

Este projeto contém notebooks Jupyter para treinamento e análise de modelos de detecção de armas de fogo usando YOLO e Faster R-CNN.

## Como Executar os Notebooks Jupyter

Para executar os notebooks Jupyter deste projeto, você pode usar o seguinte comando:

```bash
py -m jupyter notebook
```

Ou, se preferir usar o JupyterLab:

```bash
py -m jupyter lab
```

**Nota**: Se o comando `jupyter` não for reconhecido no seu terminal, use `python -m jupyter notebook` (ou `python -m jupyter lab`) que funciona mesmo quando o Jupyter não está no PATH do sistema.

Isso iniciará o servidor Jupyter e abrirá a interface no seu navegador, permitindo que você visualize e execute os notebooks disponíveis no projeto.
